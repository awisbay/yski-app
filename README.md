# Clicky Foundation App (YSKI)

Aplikasi mobile untuk Yayasan Sahabat Khairat (YSKI) yang mendigitalisasi layanan sosial yayasan.

## Fitur Utama

- **Booking Pindahan** - Pemesanan armada untuk pindahan rumah/kantor
- **Peminjaman Alkes** - Peminjaman alat kesehatan
- **Donasi / Infaq** - Penerimaan donasi dengan berbagai metode pembayaran
- **Jemput Zakat** - Penjemputan zakat ke lokasi donatur
- **Berita & Program** - Informasi kegiatan yayasan

## Tech Stack

### Backend
- FastAPI (Python 3.11+)
- SQLAlchemy 2.0 + Alembic
- PostgreSQL 16
- Redis 7
- MinIO (S3-compatible storage)

### Mobile
- React Native + Expo SDK 52
- NativeWind v4 (Tailwind CSS)
- Zustand (State Management)
- React Query (API Client)

## Quick Start

### Prerequisites
- Docker & Docker Compose
- Node.js 20+ (for mobile development)
- Python 3.11+ (for backend development)

### Development Setup

1. Clone repository:
```bash
git clone <repository-url>
cd yski-app
```

2. Copy environment file:
```bash
cp .env.example .env
# Edit .env with your settings
```

3. Start all services:
```bash
docker-compose up -d
```

4. Access services:
- API: http://localhost/api/v1
- API Docs: http://localhost/docs
- MinIO Console: http://localhost:9001

## Development

### Backend Development

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Run migrations:
```bash
alembic revision --autogenerate -m "Description"
alembic upgrade head
```

### Mobile Development

```bash
cd mobile
npm install
npx expo start
```
